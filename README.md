# **Obstacle Dodger**

> WebGL Unity Game. Drive and don't crash into the the planets!

### [Play](https://github.com/sunghogo/Obstacle-Dodger/)

### Controls:

| Key    | Controls             |
| ------ | -------------------- |
| W or ↑ | Drive Forward        |
| S or ↓ | Drive Backwards      |
| A or ← | Rotate Left          |
| D or → | Rotate Right         |
| ESC    | Quit (Only for .exe) |
